# Sistema de Empresa e Clientes em JavaScript

## 📌 Descrição

Este projeto implementa um sistema simples de cadastro de **empresa e clientes** utilizando **Programação Orientada a Objetos em JavaScript**.

A aplicação modela:

* Telefones
* Endereços
* Clientes
* Empresa

Cada cliente possui endereço e telefones, e a empresa mantém um conjunto de clientes cadastrados.

Ao executar o programa, é exibido um relatório contendo os dados da empresa e seus clientes.
